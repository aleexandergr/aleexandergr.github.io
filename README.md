## 🔧 Instalación y Uso

### Para visualizar el sitio:

1. **Opción 1: Abrir directamente**
   - Simplemente abre `index.html` en cualquier navegador moderno

2. **Opción 2: Servidor local (recomendado)**
   ```bash
   # Con Python 3
   python -m http.server 8000
   
   # O con Node.js (si tienes npx)
   npx serve
   ```
   Luego visita: `http://localhost:8000`

3. **Opción 3: Extensión VS Code**
   - Instala "Live Server" en VS Code
   - Click derecho en `index.html` → "Open with Live Server"

### No requiere instalación de dependencias
Todo funciona via CDN, no necesitas `npm install` ni ningún build.
